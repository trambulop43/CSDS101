1. Set Operations

Definitions

Union ($A \cup B$): Elements in A OR B.
Intersection ($A \cap B$): Elements in A AND B.
Difference ($A - B$): Elements in A but NOT in B.
Complement ($A'$): Elements in Universal set but NOT in A.

Cardinality Rule

$n(A \cup B) = n(A) + n(B) - n(A \cap B)$

This formula prevents double-counting the intersection.
