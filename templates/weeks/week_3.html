1. Set Operations

1.1 Union ($\cup$)

Logic: OR ($\vee$)

Combines all elements from both sets. Duplicates are listed only once.

$A \cup B = \{x \mid x \in A \lor x \in B\}$

1.2 Intersection ($\cap$)

Logic: AND ($\wedge$)

Elements that are common to BOTH sets.

$A \cap B = \{x \mid x \in A \land x \in B\}$

1.3 Difference ($-$)

"A minus B"

Elements in A that are NOT in B. It removes the overlap.

$A - B = \{x \mid x \in A \land x \notin B\}$

1.4 Complement ($'$)

Logic: NOT ($\neg$)

Everything in the Universe ($U$) that is NOT in A.

$A' = \{x \mid x \in U \land x \notin A\}$

3. Cardinality Rule (Inclusion-Exclusion)

Why can't we just add $|A| + |B|$ to find the total size? Because we would count the overlap twice!

$|A \cup B|$ = $|A|$ + $|B|$ - $|A \cap B|$

1. Add Set A

Counts the intersection once.

2. Add Set B

Counts the intersection AGAIN (Double Count!).

3. Subtract Intersection

Removes one copy of the double count.
