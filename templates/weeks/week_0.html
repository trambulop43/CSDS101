1. What is Discrete Mathematics?

It is a branch of mathematics involving discrete elements that uses algebra and arithmetic. It is increasingly being applied in the practical fields of mathematics and computer science. It is a very good tool for improving reasoning and problem-solving capabilities.

1. Continuous Mathematics

  • Based upon continuous number line or real numbers.
  • Characterized by the fact that between any two numbers, there are almost always an infinite set of numbers.
  • Example: A function plotted as a smooth curve without breaks.
  • Data: Height, Weight, Time, Temperature.

2. Discrete Mathematics

  • Involves distinct values.
  • Between any two points, there are a countable number of points.
  • Example: A list of ordered pairs, integers, binary code (0s and 1s).
  • Data: Number of students, Days of the week, Files on a PC.

2. Discrete vs. Continuous Data

Discrete Data Continuous Data
You can count the data (whole numbers). You can measure the data (scales, decimals).
Cannot be divided into smaller pieces meaningfully (e.g., 1.5 kids). Can be subdivided infinitely (e.g., 1.55 meters).
Limited number of possible values. Infinite number of possible values within an interval.
Displayed by Bar Graphs. Displayed by Histograms or Curves.

3. Real-World Applications

Computers & Files

Software and files are stored as strings of 1s and 0s. Binary math is discrete mathematics.

Networks

The internet (routers/cables) and social media (friends/followers) are modeled as discrete structures (Graphs).

Google Maps

Uses discrete math (Graph Theory) to determine the fastest driving routes and times.

Scheduling

Scheduling nurses, pilots, or rooms is solved using Graph Coloring or Combinatorial Optimization.

Cryptography

Secure shopping and data protection rely on discrete number theory (primes, modular arithmetic).

Digital Image Processing

Merging images or applying filters uses discrete matrices of pixel values.
