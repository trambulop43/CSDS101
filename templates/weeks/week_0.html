1. What is Discrete Mathematics?

It is a branch of mathematics involving discrete elements that uses algebra and arithmetic. It is increasingly being applied in the practical fields of mathematics and computer science. It is a very good tool for improving reasoning and problem-solving capabilities.

1. Continuous Mathematics

  • Based upon continuous number line or real numbers.
  • Characterized by the fact that between any two numbers, there are almost always an infinite set of numbers.
  • Example: A function plotted as a smooth curve without breaks.
  • Data: Height, Weight, Time, Temperature.

2. Discrete Mathematics

  • Involves distinct values.
  • Between any two points, there are a countable number of points.
  • Example: A list of ordered pairs, integers, binary code (0s and 1s).
  • Data: Number of students, Days of the week, Files on a PC.

2. Real-World Applications

Computers & Files

Software and files are stored as strings of 1s and 0s. Binary math is discrete mathematics.

Networks

The internet (routers/cables) and social media are modeled as discrete structures (Graphs).

Google Maps

Uses discrete math (Graph Theory) to determine the fastest driving routes and times.

Scheduling

Scheduling nurses, pilots, or rooms is solved using Graph Coloring or Combinatorial Optimization.

Cryptography

Secure shopping and data protection rely on discrete number theory (primes, modular arithmetic).

Digital Image Processing

Merging images or applying filters uses discrete matrices of pixel values.

3. Pre-Test: Diagnostic Assessment

Test your current knowledge. Click "Reveal Answer" to check.

1. Which of these is a discrete value?

a) Height of a tree
b) Number of students
c) Temperature
d) Speed of a car
Answer: b) Number of students
Discrete data is countable (integers). You cannot have 1.5 students.

2. Symbol for the set of Integers?

a) ℕ
b) ℤ
c) ℚ
d) ℝ
Answer: b) ℤ
Z stands for "Zahlen" (German for numbers).

3. In logic, what does ∧ represent?

a) OR
b) NOT
c) AND
d) IF-THEN
Answer: c) AND
The caret symbol (∧) is the logical Conjunction (AND).

4. How many bits are in a byte?

a) 4
b) 8
c) 16
d) 32
Answer: b) 8
Standard binary grouping: 1 Byte = 8 Bits.

5. Symbol for Empty Set?

a) {0}
b) 0
c) ∅
d) E
Answer: c) ∅
The symbol ∅ represents a set containing no elements.

6. Symbol for Union?

a) ∩
b) ∪
c) ⊆
d) ∈
Answer: b) ∪
Union (U-shape) combines elements from sets.

7. Which is an Irrational Number?

a) 0.5
b) 2/3
c) π
d) √4
Answer: c) π
Pi cannot be written as a simple fraction and its decimal never ends/repeats.

8. What is a Tautology?

a) Always False
b) Always True
c) Sometimes True
d) Never True
Answer: b) Always True
A statement that is true regardless of the truth values of its parts.

9. When is "If P then Q" false?

a) T → T
b) T → F
c) F → T
d) F → F
Answer: b) T → F
Implication is only false when the hypothesis is true but the conclusion is false.

10. Cardinality means:

a) Sum of elements
b) Number of elements
c) Largest element
d) Average
Answer: b) Number of elements
The size of the set. e.g., |{a,b}| = 2.

11. Probability of flipping 2 Heads (H,H)?

a) 1/2
b) 1/4
c) 1/3
d) 1/8
Answer: b) 1/4
(1/2) * (1/2) = 1/4.

12. Order matters in:

a) Permutations
b) Combinations
c) Sets
d) Groups
Answer: a) Permutations
Arrangement (order) is key in Permutations.

13. A graph with no cycles is a:

a) Tree
b) Complete Graph
c) Ring
d) Mesh
Answer: a) Tree
Trees are connected acyclic graphs.

14. Matrix Dimension Notation?

a) Rows × Cols
b) Cols × Rows
c) Length × Width
d) X × Y
Answer: a) Rows × Cols
Standard matrix notation is R x C.

15. What is 0! (Zero Factorial)?

a) 0
b) 1
c) Undefined
d) Infinity
Answer: b) 1
By definition, 0! is 1.

16. Universal Quantifier Symbol?

a) ∃
b) ∀
c) ∈
d) ∴
Answer: b) ∀
"For All" is represented by the inverted A.

17. Symbol for Subset?

a) ⊆
b) ∈
c) ∪
d) ∩
Answer: a) ⊆
"Is a subset of".

18. Matrix Inverse requires?

a) Determinant = 0
b) Determinant ≠ 0
c) Rectangular shape
d) Zero matrix
Answer: b) Determinant ≠ 0
A matrix must be non-singular to have an inverse.

19. Pascal's Triangle is used for?

a) Combinations
b) Primes
c) Geometry
d) Calculus
Answer: a) Combinations
Coefficients correspond to nCr.

20. Even number form?

a) 2k
b) 2k+1
c) 3k
d) k/2
Answer: a) 2k
Any integer divisible by 2.
