1. Deep Dive: Sets

1.1 What makes a "Set"?

A set is not just a collection; it is a well-defined collection of distinct objects.

Well-Defined (Set)

"The collection of past US Presidents."

Reason: We can objectively say if someone belongs or not.

Not Well-Defined (Not a Set)

"The collection of good movies."

Reason: Subjective. "Good" is matter of opinion.

1.2 Representing Sets

1. Roster Method (Listing)

A = {a, e, i, o, u}

Why use this?

Best for small, finite sets where you can list every element. It provides an explicit view of the data.

Details:

  • Elements are separated by commas.
  • Ellipses (...) are used for patterns (e.g., $\{1, 2, 3, ...\}$).
  • Order doesn't matter: $\{1, 2\} = \{2, 1\}$.

2. Set-Builder Notation (Rule)

A = {x | x is a vowel}

Why use this?

Essential for infinite sets (like Real numbers) or large sets where listing elements is impossible.

How to read it:

{ ... } : "The set of..."

x : "...all elements x..."

| or : : "...such that..."

P(x) : "...x meets this condition."

1.3 Cardinality ($n(A)$ or $|A|$)

The number of distinct elements in a set.

  • $A = \{2, 4, 6\}$ $|A| = 3$
  • $B = \{ \emptyset \}$ $|B| = 1$
  • $C = \emptyset$ $|C| = 0$

1.4 Equal vs. Equivalent

Equal Sets ($=$)

Must contain the exact same elements.

{1, 2} = {2, 1}

Equivalent Sets ($\sim$)

Must have the same cardinality (size).

{a, b} ~ {1, 2}

1.5 Subsets & Power Sets

A fundamental concept in discrete structures.

Subset ($\subseteq$)

Set A is a subset of B if every element of A is also in B.

Proper Subset ($\subset$)

A is inside B, but A $\neq$ B (B has at least one extra element).

Power Set $P(A)$

The set of ALL possible subsets of A, including $\emptyset$ and A itself.

Power Set Example

Let $A = \{1, 2\}$. Find $P(A)$.

Formula: Cardinality is $2^n$. Here $2^2 = 4$ subsets.

$\emptyset$ $\{1\}$ $\{2\}$ $\{1, 2\}$

2. In-Depth: Laws of Logic & Sets

Logic and Set Theory are isomorphic. This means they follow the exact same rules, just with different symbols. Understanding "The Process" helps you memorize these laws intuitively.

1. Identity Law

Basics

LOGIC

$p \wedge T \equiv p$

$p \vee F \equiv p$

SETS

$A \cap U = A$

$A \cup \emptyset = A$

The Process: Interacting with the "Identity Element" changes nothing.

  • Adding 0 to a number doesn't change it. Similarly, Union with Empty Set ($\cup \emptyset$) keeps the set the same.
  • Multiplying by 1 doesn't change it. Similarly, Intersection with Universe ($\cap U$) keeps the set the same.

2. Domination Law

Basics

LOGIC

$p \vee T \equiv T$

$p \wedge F \equiv F$

SETS

$A \cup U = U$

$A \cap \emptyset = \emptyset$

The Process: One element is so strong it "dominates" the result.

Example: If you combine your bucket ($A$) with the entire Ocean ($U$), you just have the Ocean ($U$). If you look for common elements between $A$ and Nothing ($\emptyset$), you get Nothing ($\emptyset$).

3. Idempotent Law

Simplification

LOGIC

$p \vee p \equiv p$

SETS

$A \cup A = A$

The Process: Repeating yourself adds no new information.

"It is raining OR it is raining" just means "It is raining". Combining a set with itself yields the same set.

4. De Morgan's Laws

Crucial

LOGIC

$\neg(p \wedge q) \equiv \neg p \vee \neg q$

SETS

$(A \cap B)' = A' \cup B'$

The Process: Distributing the Negative flips the Operator.

Think of it like this: "I did NOT have (Burger AND Fries)" means "I didn't have a Burger OR I didn't have Fries" (or both). The AND becomes an OR.

5. Distributive Law

Expansion

LOGIC

$p \vee (q \wedge r) \equiv (p \vee q) \wedge (p \vee r)$

SETS

$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$

The Process: Just like algebra: $a \times (b + c) = (a \times b) + (a \times c)$.

The outer operator ($\cup$) is distributed to both inner terms, and the inner operator ($\cap$) moves to the middle.

6. Double Negation

Basics

LOGIC

$\neg(\neg p) \equiv p$

SETS

$(A')' = A$

The Process: Two negatives make a positive.

"I am NOT NOT hungry" means "I am hungry". The complement of a complement is the original set.
